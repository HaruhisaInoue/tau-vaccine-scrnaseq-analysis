# Single-cell RNA-seq Analysis of Tau Vaccine-Treated Tauopathy Mouse Model

## Overview

This repository contains R scripts for analyzing single-cell RNA-seq data from a tauopathy mouse model after tau vaccine treatment.

## Analysis

The scripts are used to analyze gene expression changes and cellular responses associated with tau vaccine treatment in the tauopathy mouse model.

## Requirements

* R
* Seurat

## Usage

Run the R script in R or RStudio.

```r
source("scRNAseq_TauVac.R")
```

## Data

Single-cell RNA-seq data is not included in this repository.
